# camoservedevops

CamOservDEVOPS is the portfolio and platform showcase for Cameron “Camo” de Vries. This site highlights secure platform engineering, operational visibility, and marketplace/AI-enabled service systems across CamodevOps, 79Ratio, Buddies Worldwide, OpenClaw, OCIU, and WeCreate.

## What this project includes

- Live dashboard with GitHub repo analytics and site uptime checks
- Embedded platform intelligence infographic
- Portfolio sections for About, Experience, Projects, Ventures, Skills, and Contact
- Dark neon theme with system HUD styling
- Integration with GitHub API and external service monitoring

## Run locally

```bash
pnpm install
pnpm dev
```

## Deployment

Deploy with Vercel or any Next.js-compatible platform.
